Лабораторна робота №5
Симетричне шифрування повідомлень на основі пароля

Дисципліна: Безпека програм та даних

📌 Опис
Програма реалізує симетричне шифрування та розшифрування текстових повідомлень із використанням пароля користувача.
Криптографічний ключ формується з пароля за допомогою алгоритму PBKDF2-HMAC (SHA-256) із використанням salt та багаторазового хешування (50 000 ітерацій). Після генерації ключа для шифрування даних застосовується алгоритм Fernet (AES + HMAC).

Програма працює у консольному режимі та має інтерактивне меню для вибору дій.

🎯 Мета роботи
ознайомитися з принципами симетричного шифрування;
реалізувати безпечне формування ключа з пароля;
дослідити роль salt та багаторазового хешування;
навчитися шифрувати й розшифровувати текстові дані.

⚙️ Функціональні можливості
Програма дозволяє:
Генерувати симетричний ключ із пароля користувача;
Використовувати salt для підвищення стійкості до атак;
Шифрувати текстові повідомлення;
Розшифровувати повідомлення за умови введення правильного пароля;
Обробляти помилки при використанні неправильного ключа;
Виконувати кілька операцій без перезапуску програми.

▶️ Як запустити програму
🔹 Вимоги
Python 3.x
бібліотека cryptography
Встановлення бібліотеки:
pip install cryptography

🔹Запуск:
Збережіть код у файл, наприклад:
password_encryption.py
Запустіть програму:
python password_encryption.py

Далі користуйтеся меню в консолі.

🧠 Логіка роботи програми
create_key() — формує криптографічний ключ із пароля за допомогою PBKDF2-HMAC (SHA-256, salt, 50 000 ітерацій).
encrypt_text() — шифрує текстове повідомлення алгоритмом Fernet.
decrypt_text() — розшифровує повідомлення та обробляє помилки доступу.
main() — реалізує інтерактивне меню та керує виконанням програми.

⚠️ Примітка
Програма має навчальний характер і демонструє базові принципи захисту даних.
У реальних системах рекомендується використовувати випадкову сіль та додаткові механізми управління ключами.
