Лабораторна робота №1
Аналіз надійності пароля

📌 Опис
Дана програма призначена для оцінювання рівня безпеки пароля на основі базових критеріїв інформаційної безпеки.
Вона аналізує пароль користувача, враховуючи його довжину, склад, а також наявність персональних даних, і надає числову оцінку та рекомендації щодо покращення.
Програма реалізована мовою Python та працює у консольному режимі.

🎯 Мета роботи
навчитися оцінювати надійність паролів;
ознайомитися з базовими правилами створення безпечних паролів;
реалізувати перевірку пароля з використанням регулярних виразів;
отримувати зрозумілий зворотний зв’язок для користувача.

⚙️ Функціональні можливості
Програма перевіряє:
мінімальну довжину пароля (не менше 8 символів);
наявність великих і малих літер;
наявність цифр;
наявність спеціальних символів (!@#$%^&*);
відсутність імені користувача в паролі;
відсутність частин дати народження.
На основі перевірок:
формується оцінка від 0 до 10 балів;
визначається рівень безпеки (низький / середній / високий);
виводяться рекомендації щодо покращення пароля.

▶️ Як запустити програму
Переконайтесь, що у вас встановлений Python 3.x.
Збережіть код у файл, наприклад password_check.py.
Запустіть програму з терміналу або середовища розробки:
python password_check.py

Дотримуйтесь підказок у консолі.

🧠 Логіка роботи програми
Основна перевірка виконується у функції check_password_strength(), яка повертає оцінку та список рекомендацій.
Функція security_level() визначає рівень безпеки пароля за отриманою оцінкою.
Функція main() забезпечує взаємодію з користувачем через меню та дозволяє виконувати перевірку кілька разів без перезапуску програми.
