Лабораторна робота №2
Аналіз класичних методів шифрування

Дисципліна: Безпека програм та даних

📌 Опис
Дана програма реалізує класичні методи шифрування — шифр Цезаря та шифр Віженера — з підтримкою українського та англійського алфавітів.
Вона призначена для демонстрації принципів роботи симетричних шифрів, а також для порівняння їхньої стійкості з точки зору інформаційної безпеки.
Програма працює у консольному режимі та дозволяє виконувати як шифрування, так і розшифрування текстових повідомлень.

🎯 Мета роботи
дослідити принципи роботи класичних алгоритмів шифрування;
реалізувати шифри Цезаря та Віженера програмними засобами;
ознайомитися з поняттями ключа та зсуву;
порівняти рівень криптостійкості різних методів шифрування.

⚙️ Функціональні можливості
Програма:
підтримує український та англійський алфавіти;
автоматично формує зсув для шифру Цезаря на основі дати народження;
генерує ключ для шифру Віженера з прізвища користувача;
зберігає регістр літер під час шифрування;
ігнорує символи, що не належать до алфавітів (цифри, пробіли, знаки пунктуації);
виконує шифрування та зворотне розшифрування повідомлень.

▶️ Як запустити програму
Переконайтесь, що встановлено Python 3.x.
Збережіть код у файл, наприклад classic_ciphers.py.
Запустіть програму з терміналу або середовища розробки:
python classic_ciphers.py

Введіть необхідні дані за запитом програми.

🧠 Логіка роботи програми
calc_shift_from_date() — обчислює зсув для шифру Цезаря на основі цифр дати народження.
prepare_key_from_surname() — формує ключ для шифру Віженера з літер прізвища.
caesar_cipher() — реалізує шифрування та розшифрування методом Цезаря.
vigenere_cipher() — реалізує шифрування та розшифрування методом Віженера.
run() — забезпечує взаємодію з користувачем і виведення результатів.

🔍 Порівняння методів
Шифр Цезаря — простий у реалізації, але має низьку криптостійкість і легко піддається перебору.
Шифр Віженера — більш стійкий за рахунок використання ключового слова та змінного зсуву.
