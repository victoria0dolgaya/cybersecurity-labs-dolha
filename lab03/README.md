Лабораторна робота №3
Приховування інформації у зображеннях (стеганографія)
Дисципліна: Безпека програм та даних

📌 Опис
Даний застосунок реалізує стеганографічний метод приховування текстових повідомлень у цифрових зображеннях із використанням алгоритму LSB (Least Significant Bit).
Метод полягає у зміні найменш значущих бітів каналів RGB кожного пікселя, що практично не впливає на візуальне сприйняття зображення.

Програма дозволяє:
приховувати текст у зображенні;
витягувати приховане повідомлення;
аналізувати зміни у зображенні після вбудовування

🎯 Мета роботи
Ознайомитися з принципами стеганографії; 
реалізувати метод LSB програмними засобами без використання готових stego-бібліотек; 
навчитися приховувати інформацію без зміни візуальної якості зображення.

⚙️ Функціональні можливості
Програма:
працює з растровими зображеннями у форматі PNG;
приховує текстові повідомлення у каналах R, G, B;
зберігає довжину повідомлення у перших 32 бітах зображення;
перевіряє місткість зображення перед вбудовуванням;
підтримує інтерактивне меню (приховування / витягування);

проводить аналіз змін:
кількість змінених пікселів;
кількість змінених кольорових каналів;
порівняння розміру файлів до та після модифікації.

▶️ Як запустити програму
Вимоги:
Python 3.x
бібліотека Pillow

Встановлення:
pip install pillow

Запуск:
python stego_lsb.py

Далі використовується меню в консолі.

🧠 Логіка роботи програми
bytes_to_bits() — перетворює байти у бітову послідовність
bits_to_bytes() — відновлює байти з бітів
hide_message() — 
  перетворює текст у байти UTF-8
  додає 32 біти довжини повідомлення
  замінює молодші біти каналів RGB
  зберігає нове PNG-зображення
extract_message() —
  зчитує перші 32 біти (довжину повідомлення)
  відновлює відповідну кількість бітів
  декодує текст
print_analysis() — виконує аналіз змін у зображенні
main() — реалізує меню взаємодії з користувачем
