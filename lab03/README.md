Приховування інформації у зображеннях (стеганографія)
Дисципліна: Безпека програм та даних

📌 Опис
Дана програма реалізує стеганографічний метод приховування текстових повідомлень у цифрових зображеннях з використанням алгоритму LSB (Least Significant Bit).
Метод полягає у вбудовуванні бітів повідомлення в найменш значущі біти кольорових каналів RGB, що не впливає на візуальну якість зображення.

Програма дозволяє:
приховувати текст у зображенні;
витягувати приховане повідомлення зі стегоконтейнера.

🎯 Мета роботи
ознайомитися з принципами стеганографії;
реалізувати метод LSB програмними засобами;
навчитися приховувати інформацію без зміни зовнішнього вигляду файлу;
дослідити можливості прихованої передачі даних.

⚙️ Функціональні можливості
Програма:
працює з растровими зображеннями у форматі PNG;
приховує текстові повідомлення у каналах R, G, B;
використовує спеціальний маркер завершення для коректного декодування;
перевіряє місткість зображення перед вбудовуванням повідомлення;
підтримує режими encode та decode через командний рядок.

▶️ Як запустити програму
🔹 Вимоги
Python 3.x
бібліотека Pillow
Встановлення Pillow:
pip install pillow
🔹 Запуск
Приховування повідомлення:
python steg.py encode input.png output.png Ваш текст
Витягування повідомлення:
python steg.py decode output.png

🧠 Логіка роботи програми
bytes_to_binary() — перетворює байти повідомлення у бінарну послідовність.
binary_to_bytes() — відновлює байти з бітів.
embed_text() — вбудовує бітову послідовність у найменш значущі біти пікселів зображення.
reveal_text() — зчитує біти з пікселів і відновлює прихований текст.
STOP_SEQUENCE — спеціальний маркер, який визначає кінець прихованого повідомлення.
main() — обробляє аргументи командного рядка та керує режимами роботи програми.
